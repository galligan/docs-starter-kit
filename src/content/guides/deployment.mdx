# Deployment

Learn how to build and deploy your Docs Starter Kit site to various platforms.

## Build Process

First, build your site for production:

```bash
pnpm run build
```

This will:
- Compile your TypeScript code
- Process your MDX files
- Generate static HTML pages
- Optimize assets for production

<Alert className="mt-4">
  <AlertTitle>Build Output</AlertTitle>
  <AlertDescription>
    The build creates an optimized production build in the `.next` folder, ready for deployment.
  </AlertDescription>
</Alert>

        <CardTitle>Deploy to GitHub Pages</CardTitle>
        <CardDescription>Free hosting for static sites</CardDescription>
      </CardHeader>
      <CardContent>
        ### Setup Static Export
        
        1. Update `next.config.mjs`:
        
        ```javascript filename="next.config.mjs"
        export default withNextra({
          output: 'export',
          images: {
            unoptimized: true
          },
          basePath: '/your-repo-name'
        })
        ```
        
        2. Create GitHub Action:
        
        ```yaml filename=".github/workflows/deploy.yml"
        name: Deploy to GitHub Pages
        
        on:
          push:
            branches: [main]
        
        jobs:
          deploy:
            runs-on: ubuntu-latest
            steps:
              - uses: actions/checkout@v3
              - uses: pnpm/action-setup@v2
                with:
                  version: 8
              - uses: actions/setup-node@v3
                with:
                  node-version: 18
                  cache: 'pnpm'
              - run: pnpm install
              - run: pnpm run build
              - uses: peaceiris/actions-gh-pages@v3
                with:
                  github_token: ${{ secrets.GITHUB_TOKEN }}
                  publish_dir: ./out
        ```
      </CardContent>
    </Card>
  </TabsContent>

## Post-Deployment

### Setting Up Search

For static exports, set up Pagefind for search:

<Card className="mt-6">
  <CardContent className="pt-6">
    1. Install Pagefind:
    ```bash
    pnpm add -D pagefind
    ```
    
    2. Add post-build script:
    ```json filename="package.json"
    {
      "scripts": {
        "postbuild": "pagefind --site out --output-path out/_pagefind"
      }
    }
    ```
    
    3. Search will be available after deployment
  </CardContent>
</Card>
